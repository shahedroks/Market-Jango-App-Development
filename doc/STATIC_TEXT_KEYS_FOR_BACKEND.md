## How to use

1. Copy the blocks you need into your backend translation files (e.g. `en.php`, `fr.php`).
2. Ensure keys match exactly what the Flutter app uses in `BKeys` / `ref.t(BKeys.xxx)`.
3. Add missing keys to `lib/core/localization/Keys/buyer_kay.dart` if not already present.
4. Replace static strings in `lib/` with `ref.t(BKeys.key_name, fallback: 'Display text')`.

Keys that already exist in the app (e.g. `shipment_list`, `shipment_tab_all`, `messages`, `currency`, `settings`, `log_out`) only need to be present in the backend with the correct display value for each language.
